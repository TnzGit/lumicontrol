# Contributing

Thank you for helping improve LumiControl. Before opening a pull request:

1. Keep changes focused and explain the user-visible behavior.
2. Run the checks documented in `README.md`.
3. Do not commit credentials, private diagnostics, generated binaries, local configuration, or personal data.
4. Identify copied or adapted third-party code and preserve its copyright and license notices.
