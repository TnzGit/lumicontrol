LumiControl — контроллер яркости для Windows, реагирующий на окружающее освещение.
ESP32-C3 с датчиком GY-302/BH1750 передаёт текущую освещённость в люксах, а
экономичный фоновый Agent плавно регулирует DDC/CI-мониторы по вашей калибровочной
кривой. Дополнительная прошивка с реле позволяет управлять низковольтной подсветкой
стола или задней панели монитора. Без датчика программа рассчитывает рекомендуемую
яркость по местной погоде, высоте солнца, восходу, закату и сезонной длине дня.

## Загрузка

Последний установщик для Windows x64 доступен в разделе
**[GitHub Releases](https://github.com/TnzGit/lumicontrol/releases)**.

> [!WARNING]
> Текущие Preview-установщики пока не имеют цифровой подписи. Windows SmartScreen
> может показать предупреждение о неизвестном издателе. Перед запуском сравните
> SHA-256 с контрольной суммой, опубликованной в том же Release.

### Требования

- Windows 10 или Windows 11, x64
- монитор с включённым DDC/CI в экранном меню
- интернет и местоположение для **Weather & sun** либо ESP32-C3 SuperMini с
  датчиком GY-302/BH1750 для **USB sensor**
- совместимый модуль реле 5 В, если требуется управление подсветкой

После установки подключите USB-датчик и откройте LumiControl. Поддерживаемое
оборудование определяется автоматически; в разделе **Calibration** настройте
кривую «люксы — яркость» под помещение и каждый монитор.

## Основные возможности

- постоянно работающий экономичный Windows Agent и интерфейс Tauri по запросу
- рекомендация без оборудования по погоде, солнцу, сезону и личной поправке
- автоматическое обнаружение USB-профилей с датчиком и с датчиком плюс реле
- плавные переходы DDC/CI с изменением цели во время движения
- перетаскиваемая калибровочная кривая и история отмены на три шага
- обнаружение ручной смены яркости и отдельная калибровка мониторов
- приоритетные правила подсветки по времени, восходу, закату, погоде, люксам и
  яркости монитора
- инверсия контактов NO/NC, ручное управление и резервные действия
- светлая, тёмная и системная темы
- локальная диагностика со скрытием чувствительных идентификаторов оборудования

## Подключение оборудования

Проверенная распиновка ESP32-C3 SuperMini:

| Функция | Контакт ESP32-C3 |
| --- | --- |
| GY-302/BH1750 SDA | GPIO4 |
| GY-302/BH1750 SCL | GPIO5 |
| Вход модуля реле | GPIO10 |

Доступны профили прошивки `sensor` и `sensor-relay`. Профиль с реле рассчитан на
распространённый модуль 5 В с активным низким уровнем. Используйте модуль с драйвером
и защитой от выбросов; не подключайте катушку реле напрямую к GPIO ESP32. Сетевую
проводку необходимо изолировать от низковольтной части.

## Конфиденциальность и лицензия

Agent и интерфейс общаются через именованный канал Windows, доступный только текущему
пользователю. Настройки, резервные копии, журналы и диагностика хранятся в
`%LOCALAPPDATA%\LumiControl`. Погода запрашивается только для активных правил, которым
она нужна; учётная запись не требуется.
