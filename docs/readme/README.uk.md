LumiControl — контролер яскравості для Windows, що реагує на навколишнє світло.
ESP32-C3 із датчиком GY-302/BH1750 передає поточну освітленість у люксах, а
малоресурсний фоновий Agent плавно регулює DDC/CI-монітори за вашою калібрувальною
кривою. Додаткова прошивка з реле також може керувати низьковольтною підсвіткою столу
або задньої панелі монітора. Без датчика програма обчислює рекомендовану яскравість
за місцевою погодою, висотою сонця, сходом, заходом і сезонною тривалістю дня.

## Завантаження

Останній інсталятор для Windows x64 доступний у розділі
**[GitHub Releases](https://github.com/TnzGit/lumicontrol/releases)**.

> [!WARNING]
> Поточні Preview-інсталятори ще не мають цифрового підпису. Windows SmartScreen
> може показати попередження про невідомого видавця. Перед запуском звірте SHA-256
> із контрольною сумою, опублікованою в тому самому Release.

### Вимоги

- Windows 10 або Windows 11, x64
- монітор з увімкненим DDC/CI в екранному меню
- інтернет і місцеположення для **Weather & sun** або ESP32-C3 SuperMini з
  датчиком GY-302/BH1750 для **USB sensor**
- сумісний модуль реле 5 В, якщо потрібне керування підсвіткою

Після встановлення підключіть USB-датчик і відкрийте LumiControl. Підтримуване
обладнання визначається автоматично; у розділі **Calibration** налаштуйте криву
«люкси — яскравість» для приміщення та кожного монітора.

## Основні можливості

- постійний малоресурсний Windows Agent та інтерфейс Tauri на вимогу
- рекомендація без обладнання за погодою, сонцем, сезоном і особистою поправкою
- автоматичне виявлення USB-профілів із датчиком і з датчиком плюс реле
- плавні переходи DDC/CI зі зміною цілі під час руху
- калібрувальна крива з перетягуванням і трирівнева історія скасування
- виявлення ручної зміни яскравості й окреме калібрування моніторів
- пріоритетні правила підсвітки за часом, сходом, заходом сонця, погодою, люксами
  та яскравістю монітора
- інверсія контактів NO/NC, ручне керування та резервні дії
- світла, темна й системна теми
- локальна діагностика з приховуванням чутливих ідентифікаторів обладнання

## Підключення обладнання

Перевірена схема ESP32-C3 SuperMini:

| Функція | Контакт ESP32-C3 |
| --- | --- |
| GY-302/BH1750 SDA | GPIO4 |
| GY-302/BH1750 SCL | GPIO5 |
| Вхід модуля реле | GPIO10 |

Доступні профілі прошивки `sensor` і `sensor-relay`. Профіль із реле розрахований на
поширений модуль 5 В з активним низьким рівнем. Використовуйте модуль із драйвером та
захистом від викидів; не підключайте котушку реле безпосередньо до GPIO ESP32. Мережеву
проводку потрібно ізолювати від низьковольтної частини.

## Конфіденційність і ліцензія

Agent та інтерфейс спілкуються через іменований канал Windows, доступний лише поточному
користувачу. Налаштування, резервні копії, журнали й діагностика зберігаються в
`%LOCALAPPDATA%\LumiControl`. Погода запитується лише для активних правил, яким вона
потрібна; обліковий запис не потрібен.
